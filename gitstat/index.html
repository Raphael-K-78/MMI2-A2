Bienvenue sur GitStat

GitStat est un outil puissant qui permet d'analyser les statistiques d'un projet Git en fournissant des informations détaillées sur les commits au fil du temps. Il offre une vue d'ensemble sur les types de modifications effectuées, qu'il s'agisse d'ajouts, de suppressions ou de changements de fichiers, et permet de visualiser l'évolution de ces données dans le temps. En plus de cela, GitStat permet également d'identifier les langages de programmation les plus utilisés dans un repository, offrant ainsi une vision complète de la composition technique du projet. Que ce soit pour les développeurs cherchant à suivre leur travail ou pour les équipes analysant les tendances d'un projet open-source, GitStat constitue un outil indispensable pour une gestion optimale du code source.
